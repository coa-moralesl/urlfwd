URL Redirector

How to Use This Page

  1. Select the type of input you want to generate a URL for by choosing from the dropdown menu: TEL, URL, or UNC.
  2. Enter your input in the provided text field.
  3. Click the Generate URL button to create the redirect URL.
  4. The generated URL will automatically be copied to your clipboard and will also appear in the output field below.
  5. You can click Test URL to open it in a new tab.
